# Self-Run Vibe Bootcamp ⚡

> A free, self-directed AI-native vibe coding bootcamp. The same curriculum as the LA TechWeek Vibe Coding Camp (Oct 15, 2026), but with a free stack and a 4-month head start.

🌐 **Live site:** https://djphamdev.github.io/self-run-vibe-bootcamp

## What is this?

The LA TechWeek Vibe Coding Camp is invite-only, in LA, on Oct 15, 2026. This is the **self-run, free, online, start-now** version. Same modules, same skills, free tools.

**Built with:** Hermes Agent + OpenRouter (free models) + GitHub Pages + Cloudflare + Convex. Total cost: **$0 to start, &lt;$60/yr.**

## Local development

This is a Jekyll site (GitHub Pages native).

```bash
# Install Jekyll
gem install bundler jekyll
# Serve locally
bundle install
bundle exec jekyll serve
# Open http://localhost:4000
```

Or just edit the Markdown files and push — GitHub Pages rebuilds automatically.
