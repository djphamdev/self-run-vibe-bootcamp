          <li><strong>Prompt-first iteration</strong> — start with a prompt, refine into code</li>
          <li><strong>Real-time architecture thinking</strong> — think out loud with the model</li>

      <div class="card">
        <h3 class="text-danger">✗ Vibe Coding is not</h3>
        <ul>
          <li><strong>Blind prompt-and-pray</strong> — you still need to read the code</li>
          <li><strong>A replacement for fundamentals</strong> — context matters more than ever</li>
          <li><strong>Magic</strong> — it's a workflow, not a wish-granter</li>
          <li><strong>One tool</strong> — tool-agnostic by design, you use many</li>
          <li><strong>For solo founders only</strong> — great for teams that ship fast together</li>
          <li><strong>Free of judgment</strong> — you still need taste, you just iterate faster</li>
        </ul>
